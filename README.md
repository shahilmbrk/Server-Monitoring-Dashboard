# 🚀 Server Monitoring Dashboard

A real-time system monitoring dashboard built with Python and Flask.

## Features

- Live CPU monitoring
- RAM monitoring
- Disk usage monitoring
- Network statistics
- System uptime
- Top 5 running processes
- Responsive web dashboard
- Progress bars with dynamic updates

## Technologies Used

- Python
- Flask
- psutil
- HTML
- CSS
